# Music Genre Classification

> It is an applictaion used to classify among different music genres using supervised machine learning algorithm.

### Supported music genres
- Blues
- Classical
- Country
- Jazz
- Rock
- Pop
- Hiphop
- Metal
- Reggae
- Disco
